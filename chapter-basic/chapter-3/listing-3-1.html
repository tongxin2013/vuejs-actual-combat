表达式过长或逻辑更为复杂是应使用计算属性

<div id="app"> {{reversedText}} </div> <script> var app = new ({ el: '#app', data: { text: '123,456' }, computed: { reversedText: function () { // 这里的this指向当前Vue实例 return this.text.split(',').reverse().join(','); } } }) </script>
